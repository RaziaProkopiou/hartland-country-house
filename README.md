## Getting Started

To run the website locally for development or testing:

1. Clone the repository:
   ```
   git clone https://github.com/RaziaProkopiou/hartland-country-house.git
   ```
2. Open `index.html` in your browser (no server required for static viewing).

For the original Node.js setup (if needed):

- Ensure Node.js is installed.
- Run `npm install` (though not required for the demo).
- Start the server with `node server.js` and visit `http://localhost:3000`.
